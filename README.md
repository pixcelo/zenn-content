# Zenn Content

このリポジトリは、Zennで公開する記事と本を管理するためのものです。

## 構成

```
.
├── articles/     # 記事
├── books/        # 本
└── README.md
```

## 使い方

### 記事の作成
```bash
npx zenn new:article
```

### 本の作成
```bash
npx zenn new:book
```

### プレビュー
```bash
npx zenn preview
```
